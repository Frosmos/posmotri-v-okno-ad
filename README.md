# Проектная работа "Посмотри в окно"

**Ссылка на репозиторий:** https://github.com/Frosmos/posmotri-v-okno-ad

Этот проект выполнен в рамках обучения в Яндекс.Практикуме. Проект представляет собой веб-приложение, которое дает возможность "посмотреть в окно" в любом из выбранных пользователем городов.

## Функциональность

- Чекбокс с возможностью выбирать время суток.
- Возможность ввести в форму название интересующего города.
- Скролватор с вариантами городов.
- Кнопка "Посмотреть еще" для продления списка городов.

## Технологии
- HTML
- CSS
- JavaScript
- API
